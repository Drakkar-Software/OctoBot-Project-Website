One file and your OctoBot starts

1. Go to OctoBot's official releases page and download the latest version for your system: Windows, Linux or MacOS.

Downloadable executables for the 0.4.0-beta8 version

2. Start the downloaded executable. OctoBot opens a terminal and installs its configuration environment and the OctoBot management interface opens automatically.

Your OctoBot is ready
