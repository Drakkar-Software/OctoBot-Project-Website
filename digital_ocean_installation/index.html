Déployez votre OctoBot en cloud sur DigitalOcean.com

Déployez facilement votre OctBot dans le cloud de Digital Ocean pour qu'il soit en ligne 100% du temps.

1. Si vous n'en avez pas déjà un, créez un compte Digital Ocean. Bénéficiez de 60 jours d'hébergement gratuits en utilisant ce lien.

2. Déployez la dernière version d'OctoBot via ce lien:

3. Le déploiement de votre OctoBot commence, le processus dure une dixaine de minutes

Exemple de build d'un déploiement d'OctoBot sur Digital Ocean

4. Accédez à l'interface de management de votre OctoBot via le lien de votre application Digital Ocean, ici "https://octobot-zoctx.ondigitalocean.app/"

Administrez votre OctoBot directement avec le terminal intégré à Digital Ocean.

Le déploiement sur Digital Ocean utilise Docker. Pour administrer votre OctoBot, utilisez la console de votre application Digital Ocean: vous serez connecté directement dans votre conteneur Docker OctoBot.
