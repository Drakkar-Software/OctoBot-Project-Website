Comment utiliser OctoBot à son plein potentiel

Installer OctoBot

Plusieurs méthodes permettent de mettre en place OctoBot. Le plus simple est d'utiliser OctoBot Cloud pour lancer votre robot sur nos serveurs. Vous pouvez l'essayer gratuitement.

En utilisant OctoBot Cloud, votre OctoBot sera toujours en ligne, accessible de partout et bénéficiera des extensions suivantes:
  • Webhook intégré pour les alertes venant par exemple de tradingview.com.
  • La vue strategy designer pour obtenir des statistiques avancées sur les résultats de vos tests historiques bientôt disponible
Vous pouvez aussi directement télécharger l'exécutable, l'image Docker ou le code source et exécuter votre OctoBot chez vous.

Configurer le mode de trading à utiliser

OctoBot propose différents modules de trading : les trading modes. Chaque trading mode est unique et propose une approche différente du trading. Les trading modes peuvent:
  • Utiliser des statistiques et analyses techniques pour trouver les meilleurs entrées et sorties lors d'un trade.
  • Utiliser des algorithmes de market making pour minimiser les risques et générer des gains modestes mais réguliers.
  • Trader sur réception de signaux provenant de plateformes comme tradingview.com.
  • Suivre les meilleures stratégies provenant d' OctoBot Cloud.
  • Faire quelque chose de totalement différent basé sur vos idées: codez et testez les en utilisant OctoBot Pro.

Optimisez votre OctoBot avec le backtesting

Grâce au backtesting, éprouvez votre OctoBot sur les données historiques des marchés que vous tradez. La plupart des configurations d'OctoBot est testable en backtesting ! Optimisez la configuration de votre OctoBot dans les moindres détails pour maximiser vos profits.

Voir OctoBot docs pour plus de détails sur le backtesting.

Allez plus loin en utisant OctoBot pro

Donnez vie à vos inspirations de stratégies en utilisant OctoBot Pro.
Codez directement vos stratégies en python tout en bénéficiant de tous les outils inclus dans OctoBot et un système de rapport et de visualisation très flexible. Analysez avec précision l'exécution de vos idées sur des données historiques et créez vos meilleurs stratégies.

Partagez les stratégies de trading

OctoBot permet de facilement partager des stratégies de trading.

Pour utiliser une stratégie existante, vous pouvez:
  • Suivre une stratégie OctoBot Cloud et reproduire automatiquement chaque trade de cette stratégie
  • Importer le profile d'une stratégie pour l'utiliser ou la personaliser
Pour partager une stratégie, vous pouvez:
  • Configurer votre OctoBot pour émettre des signaux que les autres utilisateurs peuvent suivre sur OctoBot Cloud pour reproduire vos trades
  • Exporter et partager vos profils de stratégie pour permettre à d'autres utilisateurs de les utiliser
