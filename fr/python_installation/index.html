Installer OctoBot directement depuis les sources Python

Déployez votre OctoBot depuis le code afin de pouvoir le lancer dans un environnement Python local, modifier ou examiner le robot.

1. Si vous n'avez pas déjà Python, installez Python en version 3.8.

2. Clonez le dépôt OctoBot

  • git clone https://github.com/Drakkar-Software/OctoBot

3. Installez les dépendances Python du robot

  • cd OctoBot
  • python3 -m pip install -Ur requirements.txt
  • python3 start.py tentacles --install --all

4. Lancez votre OctoBot via la commande:

  • python3 start.py

L'interface de management de votre OctoBot est accessible sur le port 5001 de votre système. http://localhost:5001/ ou http://ip-address:5001/
